# fleet-management-app

A simple fleet tracking dashboard with login authentication, vehicle management, and filters.  
Built using **HTML, CSS, and JavaScript**.

---

## Features

### 1. Login Page
- Email & password input  
- Valid login → redirects to Admin Dashboard  
- Invalid login → alert message

---

### 2. Admin Dashboard

#### Add Fleet Form
- Reg No  
- Category (Auto, Car, Truck, Bus)  
- Driver Name  
- Availability (Available / Unavailable)  
- Add Fleet button  

#### Fleet Cards
Each card displays:
- Vehicle image  
- Reg No  
- Category  
- Driver Name  
- Availability Status  
- Buttons:  
  - Update Driver  
  - Change Availability  
  - Delete Vehicle  

---

### 3. Filters
- Filter by Category  
- Filter by Availability  
- Clear Filter button  
- Supports combined filtering
